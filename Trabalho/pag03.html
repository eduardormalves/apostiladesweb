Terceira aula.

Nessa aula, tivemos a introdução ao CSS, que é colocar cor, tamanho de texto, margens, entre outros no arquivo HTML.

Aprendemos duas formas de fazer isso:

  1. Usando a tag style no html
  2. Criando um link no cabeçalho da página para outra página especifica do CSS (link rel="stylesheet" href="style.css")

Sobre o CSS, aprendemos nessa aula como colocar cor, cor de fundo. Por exemplo, vou mudar a cor da página para aqua, usando a tag style, e selecionando qual parte da estrutura eu quero mudar.
