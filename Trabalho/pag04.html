Aula 04

Nessa aula de CSS, aprendemos a usar bordas, e posicionamento. Como por exemplo, os comandos:

  1. padding
  2. border: ---;
  3. border-left, border right, border-buttom, border top
  4. border-style: ---;
  5. border-color: ---;

E após todo o conhecimento adquirido durante as aulas, foi passado para nós um desafio de fazer um menu, sendo que quando passar o mouse em cima, mudasse de cor. Para fazer isso, foi necessário criar toda a estrutura usando a tag a, e no css foi necessário criar uma abertura hover.
